    Схлопывание» внешних отступов

    Вертикальный отступ между двумя соседними элементами равен максимальному отступу между ними. Если отступ одного элемента равен 20px, а второго 40px, то отступ между ними будет 40px.

    Этот эффект называется эффектом «схлопывания» внешних отступов или «схлопывания» маргинов. Горизонтальные отступы между элементами просто складываются. Например, горизонтальный отступ между двумя элементами с отступами 30px будет равен 60px. Для отступов слева и справа схлопывание никогда не применяется.

    W3C Box Model

    Это стандартное значение. Свойства width и height вычисляются исключительно по контенту, и не включая margin, padding и border.

    Traditional Box Model

    Свойства width и height вычисляются по контенту, padding и border, но не включая margin.

    Padding

    Разрешается использовать одно, два, три или четыре значения, разделяя их между собой пробелом. Эффект зависит от количества значений.

    padding: 10px;

    padding: 2em 4em;

    padding: 1em 20px 2em;

    padding: 10px 10% 2em 15%;

    Свойство float

    Элемент IMG вынимается из потока, последующие блоки начинают вести себя так, как будто его нет, и заполняют освободившееся место Элемент IMG сдвигается максимально вправо(float:right):

    Строки, в отличие от блочных элементов, «чувствуют» float и уступают ему место, обтекая картинку слева

    При float:left – всё то же самое, только IMG смещается влево (или не смещается, если он и так у левого края), а строки – обтекают справа. Строки и инлайн-элементы смещаются, чтобы уступить место float. Обычные блоки – ведут себя так, как будто элемента нет.

    блок div.left-picture «обернул» картинку и текст под ней: ширина блока с float определяется по содержимому.

    блок div.left-picture «обернул» картинку и текст под ней, и растянулся на всю ширину. Это следствие того, что ширина блока с float определяется по содержимому.

    Заполнение блока-родителя

    Из-за того, что блок с float удалён из потока, родитель не выделяет под него места.

    Image 2

    Элемент с float обязан расшириться, чтобы вместить вложенные float.

    Image 2

    Добавить в родителя элемент с clear


    Image 2

    .clearfix:after { content: "."; /* добавить содержимое: "." */ display: block; /* сделать блоком, т.к. inline не может иметь clear */ clear: both; /* с обеих сторон clear */ visibility: hidden; /* сделать невидимым, зачем нам точка внизу? */ height: 0; /* сделать высоту 0, чтобы не занимал место */ } Добавив этот класс к родителю, получим тот же результат, что и выше.

    Свойство CSS overflow определяет, необходимо ли для переполненного блочного элемента содержимое обрезать, предоставить полосы прокрутки или просто отобразить.


    При помощи float можно размещать блочные элементы в строке, похоже на display: inline-block:

    

    .gallery li {
      float: left;
      width: 300px;
      list-style: none;
    }
